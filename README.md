# 📌Tobimoa
<br/>

토비모아 (Tobimoa), 불어로 Trouvez moi('나를 찾아줘!')이다.

<br/>

'토비모아'라는 이름을 가진 미아방지(위치추적) 팔찌이며,

어플리케이션을 통해 위치와 스탬프를 확인할 수 있다.
